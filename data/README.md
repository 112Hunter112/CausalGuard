# CausalGuard training & evaluation data

Generated by `python scripts/download_datasets.py` (see `trainingAdvice.txt`).

## Downloaded

- **InjecAgent/** — From [uiuc-kang-lab/InjecAgent](https://github.com/uiuc-kang-lab/InjecAgent):
  - `user_cases.jsonl` — benign user tool cases (for Layer 5 training).
  - `attacker_cases_dh.jsonl` — direct-harm attack cases.
  - `attacker_cases_ds.jsonl` — data-stealing attack cases.

HuggingFace datasets are cached by the `datasets` library (e.g. under `~/.cache/huggingface/`), not under this folder.

## Usage

- **Layer 1:** Run on attacker instructions (InjecAgent), deepset/prompt-injections, microsoft/llmail-inject-challenge.
- **Layer 5:** Train on InjecAgent benign sessions; evaluate on attacker tool sequences. Checkpoint saved to `causalguard/checkpoints/layer5_injecagent.pt`.

## Other datasets (manual)

- **ToolBench:** `load_dataset("stabletoolbench/ToolEnv2404")` or see [OpenBMB/ToolBench](https://github.com/OpenBMB/ToolBench).
- **AgentHarm:** Clone [EuropeanAIOffice/AgentHarm](https://github.com/EuropeanAIOffice/AgentHarm) for Layer 4 multi-step tool tests.
